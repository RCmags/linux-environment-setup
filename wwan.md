# WWAN Card configuration:
If the laptop/computer has a WWAN card, it's possible to receive text messages without a cellphone. When combined with `waydroid`, it's possible to more-or-less, run a system without a physical android phone.   

Tt should be noted that the `camera` will likely not work on waydroid, so this limits the functionality of certain apps.
